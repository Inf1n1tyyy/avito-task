# Веб-приложение для модерации объявлений

Решил впервые попробовать использовать MUI, без особого углубления в красоту: https://mui.com/material-ui/

В детальной информации об объявлении для отображения изображений используется библиотека Swiper: https://swiperjs.com/react

Для отображения графиков на странице статистики используется библиотека Recharts, которая предоставляет удобные решения для разных типов графиков: https://recharts.github.io/

Кнопки для модерации объявления появляются исходя из необходимости

Частично реализованы хоткеи, удалось реализовать переключение между объявлениями по нажатию стрелок и одобрение объявления по нажатию клавиши A

## Запуск сервера

cd .\avito-task\tech-int3-server\

npm install

npm run dev

## Запуск клиента

cd .\avito-task\frontend\

npm install

npm run dev
